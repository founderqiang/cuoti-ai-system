# 面向精准教学的错题归因与靶向变式智能生成系统

> 🏫 创AI案例 · 智能信息系统 · 学情智能分析系统

## 📌 系统简介

本系统是一个面向初中数学教师的**考后智能分析与精准教学辅助平台**，解决教师在月考/大考后面临的三大痛点：

1. **海量错题难以快速统计** → 一键上传Excel，自动生成知识薄弱可视化图表
2. **缺乏深度错因分析** → AI推理归因，精准定位错因并给出讲评策略
3. **手工找变式题极其耗时** → 自动生成难度递进的靶向变式训练题

## 🎯 功能模块（对应官方5大功能要求）

| 官方要求 | 本系统对应功能 |
|---------|-------------|
| 数据分析 | Excel成绩单解析，知识点错误率统计 |
| 数据可视化呈现 | 雷达图 + 柱状图 + 数据表格 |
| 推理预测 | AI错因归因（计算粗心/辅助线构造错误等） |
| 策略辅助 | 讲评策略建议（重点/方法/时间分配/作业） |
| 内容生成 | 3-5道难度递进的靶向变式训练题及详解 |

## 🚀 快速开始

### 环境要求
- Python 3.8+
- 国产大模型API密钥（推荐 DeepSeek）

### 启动步骤

1. **配置API密钥**
   ```
   复制 backend/.env.example 为 backend/.env
   填入你的大模型API密钥
   ```

2. **一键启动**
   ```
   双击运行: 启动系统.bat
   ```

3. **访问系统**
   ```
   浏览器打开: http://localhost:5000
   ```

### 手动启动（命令行）

```bash
cd 错题归因系统
pip install -r backend/requirements.txt
python backend/app.py
```
